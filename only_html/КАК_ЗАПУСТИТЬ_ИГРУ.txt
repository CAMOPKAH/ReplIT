ИНСТРУКЦИЯ ПО ЗАПУСКУ ИГРЫ "ЯБЛОКИ И ЁЖИК"

Для запуска игры:

1. Просто откройте файл "index.html" в любом современном браузере:
   - Chrome
   - Firefox
   - Edge
   - Safari
   - Opera

2. Никаких дополнительных установок или настроек не требуется.
   Игра полностью автономна и работает без сервера.

3. Рекомендуется включить звук для получения полноценного игрового опыта.

ПРАВИЛА ИГРЫ:

1. Нажимайте на яблоки на дереве, чтобы ёжик их собирал.
2. После сбора всех яблок вам нужно ответить, сколько яблок собрал ёжик.
3. При правильных ответах сложность будет постепенно увеличиваться.
4. Если вы сделаете ошибку, игра даст подсказку.

Приятной игры!
