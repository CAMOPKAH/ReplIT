Обучающая игра "Яблоки и ёжик"

Помогите ёжику собрать яблоки и научитесь считать в увлекательной форме!

Начать игру!

Как играть:

  1. Нажимайте на яблоки на дереве
  2. Ёжик будет собирать упавшие яблоки
  3. После сбора всех яблок выберите правильное число
  4. С каждыми 3 правильными ответами сложность растёт

Эта игра создана для работы без сервера и может быть запущена через двойной клик по файлу index.html
